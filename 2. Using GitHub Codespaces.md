# 2. Using GitHub Codespaces

This is the fastest way to start.

1. Create a GitHub repository.
2. Upload your Django project.
3. Click **Code → Codespaces → Create codespace on main**.
4. Wait for the environment to start.
5. Open the terminal.

Check if Docker is available:

```bash
docker --version
docker compose version
```

Create a Dockerfile (starting point for docker) and open it:

```bash
touch Dockerfile
code Dockerfile
```

Paste something like this for a Django project in the Dockerfile:

```dockerfile
FROM python:3.12-slim

WORKDIR /app

COPY requirements.txt .

RUN pip install --no-cache-dir -r requirements.txt

COPY . .

EXPOSE 8000

CMD ["python", "manage.py", "runserver", "0.0.0.0:8000"]
```

Then you can build and run docker container:

```bash
docker build -t mysite .
docker run -p 8000:8000 mysite
```

No software needs to be installed on your PC.

---

If your Dockerfile is in another folder, for example:

```
Docker-Pratice/
    backend/
        Dockerfile
```

Then run:

```bash
docker build -t mysite backend
```

or

```bash
cd backend
docker build -t mysite .
```
